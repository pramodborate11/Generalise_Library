# 📚 Generalized Data Structure Library (C++)

## 👨‍💻 Author  
Pramod Khandu Borate

---

## 📌 About Project  
This project is a Generic Data Structure Library in C++ built using templates.  
It provides reusable implementations of different types of linked lists with common operations.  
The goal of this project is to strengthen understanding of Data Structures and Generic Programming.

---

## 🚀 Data Structures Implemented  

- Singly Linear Linked List (SinglyLLL)  
- Singly Circular Linked List (SinglyCLL)  
- Doubly Linear Linked List (DoublyLLL)  
- Doubly Circular Linked List (DoublyCLL)  

---

## ⚙️ Operations Supported  

- InsertFirst()  
- InsertLast()  
- InsertAtPos()  
- DeleteFirst()  
- DeleteLast()  
- DeleteAtPos()  
- Display()  
- Count()

## 📌 Example Usage  

SinglyLLL<int> obj;

obj.InsertFirst(10);  
obj.InsertLast(20);  
obj.InsertAtPos(15, 2);  

obj.Display();   // Output: |10|->|15|->|20|->NULL
